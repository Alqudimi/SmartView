# Comprehensive User Guide

Welcome to the Smart View ecosystem. This guide covers detailed usage.

## Receiver Mode (TV)
1. **Launch:** Select 'TV (Receiver)' on startup.
2. **Settings:** Tap the gear icon. You can set a custom Device Name, toggle PIN requirements, and sync the device hotspot.
3. **Start Server:** Tap the large central button. It transitions from 'Starting' to 'Ready'.
4. **Playback Controls:** During an active cast, tapping the screen brings up an overlay allowing you to Mute, Pause, or forcefully Disconnect the sender.

## Sender Mode (Phone)
1. **Launch:** Select 'Phone (Sender)'.
2. **Discovery:** The app scans for nearby GOs (Group Owners). Available devices appear highlighted.
3. **Connection:** Tap a device to connect, or use the QR Code scanner for an instant bypass.
4. **Active Session:** During casting, a timer is displayed. Tap 'Stop Casting' to terminate the session and tear down the RTSP connection.

## Language Support
Tap the Globe icon on the role selection screen to toggle between English and Arabic (`rtl` layout support is fully implemented).
